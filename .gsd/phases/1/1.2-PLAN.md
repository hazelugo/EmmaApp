# Plan 1.2: Build CutsceneOverlay Component

## Objective
Implement the global `CutsceneOverlay.vue` that renders full-screen over the app layout, plays an assigned `.mp4`, and emits an event when the video finishes or is skipped.

    Create a new Vue SFC that acts as a fullscreen video player.
    - **Template**: A fixed full-screen wrapper (`position: fixed; inset: 0; z-index: 50`) using standard Tailwind utilities. Inside, include a `<video>` element with `autoplay`, `playsinline`, `muted`, and remove controls. The video should scale nicely (e.g. `object-cover` or `object-contain`).

      - Add an emit for `ended` event.
      - Hook the native HTML DOM `<video @ended="...">` up to fire the `emit('ended')`. 
      - Implement a `skipVideo` helper that also triggers the `emit('ended')` so the flow continues smoothly.
